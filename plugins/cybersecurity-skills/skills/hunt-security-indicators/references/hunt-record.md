# Security Hunt Record

```markdown
- Hunt question and source incident/finding:
- Assets/identities/environments/time window:
- Indicator or behavior, provenance, confidence, expiry:
- Data source, retention, delay, and permissions:
- Exact query/tool/version/timezone:
- Filters and known benign context:
- Matches and independent validation:
- False positives and rationale:
- Sources/assets not searched or failed:
- Coverage-bounded negative result:
- Approved pivots and next action:
```

Hash, address, and domain indicators decay quickly. Record date checked and prefer durable behavior/context when possible.
