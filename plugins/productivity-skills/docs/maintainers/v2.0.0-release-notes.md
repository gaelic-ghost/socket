# v2.0.0

## What changed

- Simplified the public skill surface to four standalone top-level skills:
  - `project-docs-maintainer`
  - `project-workspace-cleaner`
  - `things-reminders-manager`
  - `things-digest-generator`
- Removed the routing layer and legacy roadmap shim so the repo now presents direct install and invocation paths only.
- Recentered repo docs and maintainer docs on the canonical surviving surfaces, especially `project-docs-maintainer` for roadmap work.

## Breaking changes

- `project-skills-orchestrator-agent` has been removed.
- `project-roadmap-maintainer` has been removed.
- These install and invocation surfaces no longer exist in `v2.0.0`.

## Migration/upgrade notes

- If you previously installed or invoked `project-roadmap-maintainer`, switch to `project-docs-maintainer` and use `mode=roadmap_maintenance`.
- If you previously started with `project-skills-orchestrator-agent`, install and invoke the remaining standalone skills directly instead of routing through a front-door skill.

Canonical install commands:

```bash
npx skills add gaelic-ghost/productivity-skills --skill project-docs-maintainer
npx skills add gaelic-ghost/productivity-skills --skill project-workspace-cleaner
npx skills add gaelic-ghost/productivity-skills --skill things-reminders-manager
npx skills add gaelic-ghost/productivity-skills --skill things-digest-generator
```

Roadmap maintenance replacement:

```text
Use $project-docs-maintainer with mode=roadmap_maintenance
```

Standalone install-all command:

```bash
npx skills add gaelic-ghost/productivity-skills --all
```
