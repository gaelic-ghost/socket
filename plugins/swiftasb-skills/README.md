- `build-swift-package`
- `diagnose-integration`

## Use It For

Use this plugin when an agent needs to explain SwiftASB, choose an integration shape, build SwiftASB-facing SwiftUI or AppKit code, or diagnose SwiftASB integration failures.

Use `apple-dev-skills` for Apple framework rules, Xcode workflow selection, SwiftUI/AppKit lifecycle behavior, DocC, build, and test execution.
