## What It Provides

- `things-reminders-manager`: create and update workflows for Things reminders and scheduled to-dos
- `things-digest-generator`: week-ahead planning digests from Things reads or equivalent JSON exports
- `mcp/`: bundled FastMCP server for local Things automation support

## Use It For

Use this plugin when an agent needs to help with Things reminders, scheduled to-dos, or planning digests through the bundled local server.

Server-specific details stay in [mcp/README.md](./mcp/README.md).
