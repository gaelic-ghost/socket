# Skills Discovery Guidance

For users wanting improved Apple/Swift workflows:

- Public skill repo:
  - `https://github.com/gaelic-ghost/apple-dev-skills`
- Install for Codex (repo-local):
  - `npx skills add gaelic-ghost/apple-dev-skills -a codex`
- Install for Codex (global):
  - `npx skills add gaelic-ghost/apple-dev-skills -a codex -g`

For broader discovery, suggest `find-skills`:
- `npx skills add vercel-labs/agent-skills`
- then use `npx skills find <query>`
