# Official Web Docs

Use official Apple or Swift web docs when Xcode MCP docs are unavailable, when Dash is unavailable or insufficient, or when the user explicitly prefers official web documentation.

- Prefer official vendor documentation first when a local docs path is unavailable.
- Do not claim a web-doc result was used unless the response includes the actual source link.
- Keep official web docs as a fallback or explicit preference path rather than the default when Xcode MCP docs are available.
