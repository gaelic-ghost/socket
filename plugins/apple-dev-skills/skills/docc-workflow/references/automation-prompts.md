# Automation Prompts

Use these intent anchors when shaping or evaluating the workflow:

- symbol comments: summary, discussion, parameters, returns, examples, behavior, side effects
- article work: article, overview, conceptual page, guide, supporting content
- structure work: extension file, landing page, topic group, topics section, catalog structure, organization
- tutorial-aware work: tutorial, walkthrough, guided learning, learning path, step-by-step lesson
- docs-lookup handoff: search docs, look up directive, find Apple docs, WWDC, reference page
- execution handoff: docbuild, export archive, publish docs, host docs, build documentation, archive generation
