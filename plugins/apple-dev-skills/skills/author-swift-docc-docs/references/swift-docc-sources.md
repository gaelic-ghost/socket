# Swift DocC Sources

Use the fuller Swift.org DocC material when the user needs deeper DocC syntax or structure detail than the Xcode docs provide directly.

- [DocC on Swift.org](https://www.swift.org/documentation/docc/)
- [swiftlang/swift-docc](https://github.com/swiftlang/swift-docc)
- Dash docset when installed: `swiftlang/swift-docc main`
- [swiftlang/swift-docc-plugin](https://github.com/swiftlang/swift-docc-plugin)
- [swiftlang/swift-docc-plugin on Swift Package Index](https://swiftpackageindex.com/swiftlang/swift-docc-plugin/documentation)

Reach for the fuller DocC references when the task depends on:

- deeper directive behavior
- tutorial-specific semantics
- structure rules that go beyond the lighter Xcode product guidance
- precise DocC syntax questions that should not be answered from memory
- Swift Package Manager plugin behavior for building, previewing, or integrating DocC outside the Xcode-only workflow
