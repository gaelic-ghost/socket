# Platform Matrix

## Purpose

Capture the supported first-pass platform and UI-stack combinations for `bootstrap-xcode-app-project`.

## Current Supported Execution Matrix

| Platform | UI stack | Current supported mutating path |
| --- | --- | --- |
| `macos` | `swiftui` | `xcodegen` |
| `ios` | `swiftui` | `xcodegen` |
| `ipados` | `swiftui` | `xcodegen` using the iOS platform family |

## Current Guided-Only Matrix

| Platform | UI stack | Current status |
| --- | --- | --- |
| `macos` | `appkit` | guided-only |
| `ios` | `uikit` | guided-only |
| `ipados` | `uikit` | guided-only |
