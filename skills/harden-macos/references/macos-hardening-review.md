# macOS Hardening Review

Review in risk order:

1. active compromise or exposed credentials;
2. supported OS and current security data updates;
3. FileVault, screen lock, backups, and recovery ownership;
4. administrator accounts, remote access, sharing, and listening services;
5. Gatekeeper/XProtect/SIP/TCC and security-tool health;
6. login/background items, profiles, helpers, and extensions;
7. browsers, downloads, password manager, MFA, and recovery methods;
8. monitoring, alert ownership, and periodic recheck.

Use current Apple Platform Security and official support documentation for the exact OS. Do not prescribe disabling SIP, Gatekeeper, quarantine, XProtect, or automatic security updates as routine hardening.
