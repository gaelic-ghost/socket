# Hopper MCP Profile Reference

Hopper's MCP server is a local stdio process, but its tool results become part
of the connected agent context. Select a document, procedure, address range,
or string query as carefully as any other disclosure to the active model or
provider.

Start with a small read/navigation allowlist: document discovery, segment and
procedure lookup, string and name search, address navigation, assembly,
pseudocode, callers, callees, cross-references, and read-only annotations.
Keep comments, renames, bookmarks, patches, document selection, and bulk edits
disabled until a saved checkpoint and explicit edit request exist.

Socket does not distribute Hopper or a portable launcher. Configure the
installed `HopperMCPServer` command in private Codex or Hermes configuration;
do not add a Socket `.mcp.json` declaration for it.
