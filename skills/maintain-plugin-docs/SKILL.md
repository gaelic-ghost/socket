# Maintain Plugin Docs

Maintain docs for a skills-export repository.

## Codex Limitation Warning

When this skill touches Codex guidance, warn plainly that OpenAI's documented Codex plugin system does not provide proper repo-private plugin scoping. Do not soften that warning.

## What This Skill Does

- audits `README.md`, `ROADMAP.md`, and cross-doc consistency
- tightens misleading wording
- keeps docs aligned with the live exported skills and the repo's real model

## What This Skill Does Not Do

- it does not install anything
- it does not validate repo-local plugin install surfaces
- it does not normalize nested plugin-directory guidance for this repo
