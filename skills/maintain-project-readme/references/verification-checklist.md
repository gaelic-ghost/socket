# Verification Checklist

- The README has a title and one-line value proposition.
- `## Overview` exists and contains `### Motivation`.
- `## Setup`, `## Usage`, `## Development`, `## Verification`, and `## License` are present.
- The detected profile matches the repo signals or the report calls out ambiguity.
- Command examples do not reference unsupported toolchains for the repo.
- `apply` mode changes only the target `README.md`.
- Clean runs emit exactly `No findings.`
