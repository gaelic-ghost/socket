# Allowlist Guidance

If official commands are blocked by sandbox or allowlist policy, tell the user which safe command prefix needs approval in `~/.codex/rules`.

## Typical safe prefixes

- `["xcodebuild"]`
- `["xcrun"]`
- `["swift"]`
- `["swiftly"]`
- `["xcode-select"]`

Do not suggest broad or unsafe allowlist patterns.
